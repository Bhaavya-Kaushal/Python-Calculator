📌 Python Calculator

This is a simple calculator program written in Python.
It performs basic arithmetic operations using a menu-driven approach.

✨ Features
Addition
Subtraction
Multiplication
Division
Quotient of division
Remainder of division
Exit option
🛠️ How it works
The user selects an operation from the menu
Enters two numbers
The program displays the result
▶️ How to run
Make sure Python is installed
Download or clone this repository
Open terminal / command prompt
Run the file:
python calculator.py

Python basics
Loops (while)
Conditional statements (if-elif-else)
Exception handling (try-except)
